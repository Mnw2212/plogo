# plogo
LOGO language using python.

## Supported Functions
```
FD 60
```
Takes the turtle forward by 60 units
```
BK 60
```
Takes the turtle back by 60 units
```
RT 90
```
Turns the turtle right by 90 degress
```
LT 45
```
Turns the turtle left by 45 degrees
```
EXIT
```
Exits the console.

## How to run 
```
$ python input.py
```
